# 640. Solve the Equation

Using regular expression, be careful with different cases.

```java
class Solution {
    public String solveEquation(String equation) {
        String[] parts = equation.split("=");
        int[] leftPart = evaluate(parts[0]);
        int[] rightPart = evaluate(parts[1]);
        if (leftPart[0] == rightPart[0] && leftPart[1] == rightPart[1]) {
            return "Infinite solutions";
        } else if (leftPart[0] == rightPart[0]) {
            return "No solution";
        }
        return "x=" + (rightPart[1] - leftPart[1]) / (leftPart[0] - rightPart[0]);
    }

    private int[] evaluate(String str) {
        String[] tokens = str.split("(?=[+-])");
        int[] res = new int[2]; // res[0] is coefficient of x, res[1] is coefficient of constant
        for (String token : tokens) {
            if (token.equals("+x") || token.equals("x")) {
                res[0]++;
            } else if (token.equals("-x")) {
                res[0]--;
            } else if (token.contains("x")) {
                res[0] += Integer.parseInt(token.substring(0, token.length() - 1)); // get coefficient of x
            } else {
                res[1] += Integer.parseInt(token); // all digits
            }
        }
        return res;
    }
}
```
